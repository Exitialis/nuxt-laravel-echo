## Setup
- Add `nuxt-laravel-echo` dependency using yarn or npm to your project
- Add `nuxt-laravel-echo` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-laravel-echo',

    // With options
    ['nuxt-laravel-echo', { /* module options */ }],
 ]
}
```
